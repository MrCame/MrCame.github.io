示例：nums = [-1, 0, 1, 2, -1, -4]，和为0的三元组集合 = [[-1, 0, 1], [-1, -1, 2]]。

思路：将数组排序后，指针指定一个元素后，对该元素之后的元素进行2sum运算，考虑到不能有重复结果，所以需要跳过重复元素。
